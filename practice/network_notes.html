Protocol
The set of rules that dictate how information must be formatted.
Packet
A tiny piece of information broken from the larger message. One message might contain millions of packets.
Header
Information about a packet. To, from, order, etc.
Payload
The actual message contained in a set of packets.
Router
A place that many packets from different computers will travel through.
Routing Table
A table that stores the location of routers based on their IP addresses.
